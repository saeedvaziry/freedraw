# FreeDraw

A lightweight, open-source, self-hostable web app for drawing diagrams and charts — a minimal alternative to tools like Lucidchart. No accounts, no servers: everything runs in your browser and is saved locally so a refresh never loses your work.

## Development

```bash
npm install --ignore-scripts
npm run dev
```

Other scripts: `npm run build`, `npm run test`, `npm run lint`, `npm run typecheck`.
